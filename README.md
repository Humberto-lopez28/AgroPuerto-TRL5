# AgroPuerto: Plataforma web para la comercialización directa de productos agrícolas en la vereda La Bendición (Puerto Carreño)

Este repositorio contiene el prototipo web funcional con un nivel de maduración tecnológica **TRL5**, desarrollado como propuesta ingenieril para fortalecer las competencias digitales y mitigar la intermediación comercial de los pequeños productores rurales del municipio de Puerto Carreño, Vichada.

## 💻 Stack Tecnológico Utilizado
* **Front-End (Capa de Presentación):** HTML5, CSS3, JavaScript Vanilla (Optimizado sin frameworks pesados para entornos de baja conectividad regional).
* **Back-End (Lógica del Sistema):** PHP 8.x con arquitectura de patrón de diseño Modelo-Vista-Controlador (MVC).
* **Base de Datos:** MySQL / MariaDB (Estructura relacional normalizada).

## 📁 Estructura del Repositorio (Patrón MVC)
* `/models` : Clases y lógica de conexión a la base de datos relacional.
* `/controllers` : Controladores encargados de procesar el registro y el catálogo de ofertas.
* `/views` : Interfaces responsivas bajo filosofía *Mobile-First* para el catálogo público y el módulo privado del productor.
* `/database` : Script `.sql` para la importación y creación de las tablas de la base de datos.
